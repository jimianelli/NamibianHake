# Read ADMB .psv file

Read ADMB .psv file and return an R object of type 'list'

## Usage

``` r
read_psv(fn, nsamples = 10000)
```

## Arguments

- repfile:

  name of ADMB output file to be read (no extension needed)

## Value

object of type 'list' with ADMB outputs therein
