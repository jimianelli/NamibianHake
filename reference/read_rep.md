# Read ADMB .rep file

Read ADMB .rep file and return an R object of type 'list'

## Usage

``` r
read_rep(fn)
```

## Arguments

- fn:

  name of ADMB output file to be read (no extension needed)

## Value

object of type "list" with ADMB outputs therein
