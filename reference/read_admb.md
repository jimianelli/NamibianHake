# Read ADMB output files

Read ADMB output files .rep, .par, and .cor and return an R object of
type 'list'

## Usage

``` r
read_admb(repfile)
```

## Arguments

- repfile:

  ADMB output files to be read (no extension needed)

## Value

object of type 'list' with ADMB outputs as list elements
